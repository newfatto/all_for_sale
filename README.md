# Проект All For Sale
## Описание:
All For Sale - это веб-приложение на django для 
управления интернет-магазином.
## Установка:
Клонируйте репозиторий:
```
gh repo clone newfatto/all_for_sale
```
## Функционал:
* В проекте реализовано приложение catalog, представляющее
собой сайт интернет-магазина. 
* html страницы стилизованы с 
использованием bootstrap (файлы стилей и скриптов расположены
в папке /static). 
* При нажатии на кнопки навигации ("Каталог" и "Контакты"
происходит переадресация на соответствующие страницы)
* При отправке данных через форму на странице "Контакты"
в консоль выводится информация, введённая в поля формы, а
пользователю возвращается сообщение об успешной отправке.

## Использование:
1. Запустите локальный сервер.
```
python manage.py runserver
```
2. Перейдите по адресу, появившемуся в терминале. 
Обычно это http://127.0.0.1:8000/

3. Используйте функционал сайта (перемещайтесь в разделы,
нажимая на кнопки, заполните форму на странице "Контакты")
